# Data

Este diretorio guarda dados simples e versionaveis usados pelos exemplos.

## O que fica aqui

- catalogos pequenos em JSON
- fixtures de testes
- exemplos locais para skills

## O que nao deve ficar aqui

- segredos
- dumps grandes
- bases vetoriais pesadas
- arquivos gerados em runtime

Se o projeto crescer, trate este diretorio como fonte de dados pequena e previsivel, adequada para Git.
