# Examples

Este diretorio tem exemplos pequenos para rodar rapido.

## Objetivo

- mostrar uso direto da API OpenAI-compatible
- mostrar um agente LangChain
- mostrar um workflow LangGraph

## Como usar

Ative o ambiente virtual, configure o `.env` e rode os scripts:

```bash
python -m examples.use_openai_sdk
python -m agents.support_agent
python -m workflows.support_graph
```
