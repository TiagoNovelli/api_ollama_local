# Prompts

Este diretorio guarda instrucoes versionadas para agentes.

## Por que versionar prompts

Prompts sao parte do comportamento do sistema. Se mudam, o agente muda.

Versionar prompts ajuda a:

- entender regressao de comportamento
- revisar alteracoes em pull requests
- reaproveitar instrucoes em varios agentes

## Regras praticas

- um prompt por arquivo
- texto claro, sem esconder regra em codigo
- separar papel, tom, objetivo e limites
